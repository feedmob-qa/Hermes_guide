## 🔥 为什么你需要这本书？

AI Agent 正从"能聊天"进化到"能做事"。但市面上的教程要么停留在 API 调用层面，要么直接甩给你源码。

**这本书填补了那个空白。**

Hermes Agent 是 Nous Research 出品的开源 AI 智能体框架——它不是又一个 LLM 壳，而是一套完整的 Agent 运行时系统：

```text
对话 → 工具调用 → 记忆持久化 → 多平台网关 → Cron 调度 → 看板编排
```

从你敲下第一条命令，到用看板编排 10 个子代理并行开发一个完整项目——这本书一步步带你走完。
