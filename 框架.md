## 0.3 五分钟上手

### 0.3.1 前置要求

| 类型 | 要求 |
|------|------|
| 最低 | Python >= 3.10, pip, 网络连接 |
| 推荐 | 8GB+ RAM, Linux/macOS/WSL2, API Key |

### 0.3.2 一键安装

```bash
# Linux / macOS / WSL2
curl -fsSL https://get.hermes-agent.dev | bash

# 或使用 pip
pip install hermes-agent

# Termux (Android)
pkg install hermes-agent
```

### 0.3.3 首次配置

```bash
# 运行设置向导
hermes setup

# 或手动配置 API Key
export OPENROUTER_API_KEY="sk-or-v1-xxxxx"

# 验证安装
hermes doctor
```

### 0.3.4 开始使用

```bash
# 启动交互式 TUI
hermes

# 或单次对话
hermes "你好，请介绍一下你自己"
```
